#### 对称二叉树

给定一个二叉树，检查它是否是镜像对称的。

##### 示例 1:
```$xslt
输入:          1
             / \
            2   2
           / \ / \
          3  4 4  3

        [1,2,2,3,4,4,3]

输出: true
```

##### 示例 2:
```$xslt
输入:      1
         / \
        2   2
         \   \
         3    3

        [1,2,2,null,3,null,3]

输出: false
```
